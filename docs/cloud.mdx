# Cloud

- Serverless (`function as a server`)
  - Protocals : `http://`, `https://`
- Real Time Messaging
  - AWS Simple Queue System (`SQS`)
  - Queue
  - TCP
  - kafka (`queue://`), `rabbitmq://`, `activemq://`
- Socket (provides channel)
  - No Queue
  - TCP protocol (`ws://`)
- Internal Notification service
  - AWS Short Notification System (`SNS`)
