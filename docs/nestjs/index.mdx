# Nest.js

- A progressive Node.js framework
- Building efficient, reliable and scalable server-side applications
- Built with and fully supports TypeScript
- Nest makes use of robust HTTP Server frameworks like [Express](https://expressjs.com) and optionally can be configured to use [Fastify](https://www.fastify.io) as well!
