# MongoDB

- NoSQL - Object relational database
  - No table format
  - Data form object
- Store in physical memory in Binary JSON (BJSON)
- Cloud agonistic or On-Premise
- Multi-cluster
- Shading
- Scalable
- Best choice for real time application
